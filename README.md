# ML Model Comparison Dashboard (Streamlit)

A practical ML project that trains multiple classification models and visualizes evaluation metrics in an interactive Streamlit dashboard.

## Features
- Train multiple models using a reusable evaluation utility
- Metrics: Accuracy, Precision, Recall, F1
- `metrics.json` export for tracking and reproducibility
- Streamlit dashboard:
  - Model dropdown
  - Metrics table
  - Comparison chart
  - Retrain button

## Tech Stack
- Python
- pandas
- scikit-learn
- Streamlit
- matplotlib
- joblib
